# Схема-терапія · Концептуалізація випадку

Локальний застосунок: **аудіо випадку → транскрипт → автоматична карта часток + план терапії по 3 етапах**, з ручним редагуванням і експортом у PNG / PDF.

- **Транскрипція** — Google Gemini (розуміє українську, тримає довге аудіо 15–30 хв).
- **Аналіз** — Claude (Opus 4.8) із зашитою базою знань курсу схема-терапії (18 схем, 14 часток, 3 етапи, бланк ISST).
- Працює локально у браузері. Інтернет потрібен лише для двох кнопок: «Транскрибувати» і «Аналізувати».

---

## 1. Ключі (один раз)

Відкрий файл **`.env`** у цій папці й встав два ключі:

```
GEMINI_API_KEY=...      # https://aistudio.google.com/apikey
ANTHROPIC_API_KEY=...   # https://console.anthropic.com/settings/keys
```

Без ключів усе одно працює **демо-режим** (кнопка «Демо-приклад») — щоб потренуватися заздалегідь.

## 2. Запуск

- Подвійний клік по **`Запустити.bat`** (Windows).
- Або в терміналі: `npm install` (перший раз), потім `npm start`.
- Відкриється `http://localhost:5180`.

## 3. Як користуватися на екзамені

1. У Zoom увімкни **локальний запис** (Settings → Recording → "Record a separate audio file"). Після надиктовки кейсу в папці запису буде `audio_only.m4a`.
   - Підстраховка: можна паралельно записати на диктофон телефона.
2. У застосунку: перетягни цей аудіофайл (або «Обрати файл») → **«Транскрибувати»**. За 1–3 хв з'явиться транскрипт.
3. Перевір транскрипт (можна підправити) → **«Аналізувати»**.
4. Відкриється **Карта часток** і **План терапії** — обидва редаговані:
   - клік на частку → редагуєш назву й маркери; перетягуй кружечки мишкою;
   - «➕ Додати частку» — палітра всіх часток курсу;
   - правиш «Проблеми» та «Цілі» (по одному пункту в рядку);
   - вкладка «План терапії» — 3 етапи з описом, частками й техніками.
5. Експорт: **🖼 PNG** (карта) і **🖨 Друк / PDF** (обидва листи разом).
6. «Довідник» — уся теорія курсу під рукою (схеми, частки, потреби).

> Поради: маєш цілу годину — спокійно перевір і відредагуй усе перед здачею. «💾» зберігає чернетку у файл, «📂» відкриває назад.

## Якщо аудіо не транскрибується
- Майже всі формати конвертуються автоматично (вбудований ffmpeg).
- Якщо щось пішло не так — встав транскрипт текстом вручну і одразу тисни «Аналізувати».

## Налаштування (необов'язково, у `.env`)
- `CLAUDE_MODEL=claude-opus-4-8` (найкраща якість) або `claude-sonnet-4-6` (швидше/дешевше).
- `GEMINI_MODEL=gemini-2.5-flash`.
- `PORT=5180`.
